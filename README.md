# NSMutableArray-StackExt

Simple addition to NSMutableArray which gives you a Stack implementation. Use push and pop to manage your statck for any kind of Object Stack.

Example:

    // Push objects
    [q push:@"object1"];
    [q push:@"object2"];

    // Pop objects
    NSLog(@"%@", [q dequeue] );
    NSLog(@"%@", [q dequeue] );
